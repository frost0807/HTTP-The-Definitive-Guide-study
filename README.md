- 웹개발자의 필수 지식인 HTTP에 대해 알아보자.
- 스터디를 통해 집단지성의 힘을 믿고 두꺼운 책을 완주해보자.
- 다양한 연차,직군의 개발자들과 스터디 하며 실무에서의 사례들을 공유하며 스터디 해보자.
